# hub-minijogos-c
Hub de Minijogos em C
>>>>Descrição do Projeto<<<<
Este projeto foi desenvolvido em linguagem C e consiste em um menu com três minijogos que podem ser escolhidos pelo usuário.
O objetivo do trabalho foi praticar lógica de programação, uso de estruturas de repetição, condicionais e organização de código.
## Jogos Desenvolvidos;
1. Perguntas e Respostas (quiz sobre cinema)
2. Cobra na Caixa (jogo de sorte para dois jogadores)
3. Gousmas War (jogo de estratégia por turnos)
#Linguagem Utilizada:
* Linguagem C
* Bibliotecas: stdio.h, stdlib.h, time.h, stdbool.h
##  Como Executar o Código;
Para rodar o programa, é necessário ter um compilador de C (como o GCC).
1. Baixar/clonar o repositório
2. Abrir a pasta do projeto
3. Compilar o código:
   gcc principal.c -o jogos
4. Executar:
   * Windows: jogos.exe
   * Linux/Mac: ./jogos
## Uso de Inteligência Artificial:
Durante o desenvolvimento, utilizamos a IA Gemini para entender alguns conceitos que ainda não sabíamos.
A IA nos ajudou principalmente com:
* Como usar números aleatórios (rand)
* Necessidade de adicionar bibliotecas no código
* Como funciona o switch case
* Como utilizar vetores
Essas ferramentas foram usadas apenas como apoio para aprendizado.
## O que aprendemos:
* Uso de while e do-while
* Uso de switch case
* Geração de números aleatórios
* Organização de um menu com várias opções
* Uso básico de vetores
